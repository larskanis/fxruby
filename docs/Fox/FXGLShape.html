Overview

OpenGL shape object.

Shape drawing options

SURFACE_SINGLESIDED

Single-sided [both sides same]

SURFACE_DUALSIDED

Dual-sided surface

SHADING_NONE

No light source

SHADING_SMOOTH

Smooth shaded

SHADING_FLAT

Flag shaded

FACECULLING_OFF

No face culling

FACECULLING_ON

Cull backward facing surfaces

STYLE_SURFACE

Draw filled surfaces

STYLE_WIREFRAME

Draw wire frame

STYLE_POINTS

Draw as points

STYLE_BOUNDBOX

Draw bounding box
